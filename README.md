# 📊 HR Analytics Dashboard

## 🔹 Overview

The HR Analytics Dashboard is a Power BI project designed to analyze employee data and provide meaningful insights to support HR decision-making.
This dashboard focuses on **attrition analysis, employee distribution, and performance trends**.

---

## 🎯 Objectives

* To analyze employee attrition rates
* To understand department-wise workforce distribution
* To evaluate the impact of salary and experience on employees
* To support data-driven HR decisions

---

## 🛠️ Tools & Technologies

* Microsoft Power BI
* Microsoft Excel
* Data Visualization Techniques

---

## 📁 Dataset Information

The dataset includes the following fields:

* Employee ID
* Name
* Age
* Gender
* Department
* Job Role
* Salary
* Attrition (Yes/No)
* Years at Company

---

## 📊 Dashboard Features

* 🔢 Total Employees (KPI Card)
* 📉 Attrition Rate Analysis
* 🏢 Department-wise Employee Count
* 👨‍👩‍👧 Gender Distribution (Pie Chart)
* 💰 Salary Analysis
* 📈 Experience vs Attrition Trends

---

## 🔍 Key Insights

* Higher attrition observed in certain departments
* Employees with lower experience show higher attrition
* Salary has a noticeable impact on employee retention
* Majority of employees fall within a specific age group

---

## ✅ Advantages

* Easy-to-understand visual insights
* Faster and better decision-making
* Improved HR planning
* Interactive dashboard experience

---

## ⚠️ Limitations

* Depends on data accuracy
* No real-time data integration
* Does not include predictive analytics

---

## 🚀 Future Enhancements

* Machine learning-based attrition prediction
* Real-time data connectivity
* Advanced filters and drill-down analysis

---

## 📌 Conclusion

The HR Analytics Dashboard is an effective tool for transforming employee data into actionable insights.
It helps HR teams improve hiring strategies, reduce attrition, and make better workforce decisions.
